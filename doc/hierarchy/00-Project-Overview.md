# Project Overview

Production ASP.NET Core 10 E-commerce using Microservice Architecture.

## Microservices
- API Gateway
- Identity Service
- Catalog Service
- Inventory Service
- Cart Service
- Checkout Service
- Order Service
- Payment Service
- Shipping Service
- Notification Service
- Reporting Service

Communication:
- REST (sync)
- Message Broker (async)
- JWT authentication
